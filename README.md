# Repflow Fitness Website

## 📌 Description
Repflow is a fitness-based web application that helps users manage their weight, track calories, and follow structured diet and workout plans. It provides personalized calorie intake suggestions based on user goals such as calorie deficit or surplus.

## 🚀 Features
- 🔢 Calculates current weight and suggests calorie intake
- 📉 Supports calorie deficit and calorie surplus goals
- 🥗 Displays diet chart with food items and their macronutrients
- 🧮 Macro calculator for tracking protein, carbs, and fats
- 💪 Workout section with muscle-specific exercise videos (biceps, chest, etc.)
- 📞 Contact page for enrolling in a 6-month personal training program

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
